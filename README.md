# 🌫️ Kanagawa Mist for Zed

![Kanagawa Mist Theme](https://img.shields.io/badge/Theme-Kanagawa%20Mist-blueviolet)

A fork of [ethangilmore/zed-kanagawa](https://github.com/ethangilmore/zed-kanagawa), rebranded as **Kanagawa Mist**, with **muted border colors** for a softer, less distracting UI.

Repository: [hisea/zed-kanagawa-mist](https://github.com/hisea/zed-kanagawa-mist)

## About

This extension is a Zed port of [rebelot's Kanagawa.nvim](https://github.com/rebelot/kanagawa.nvim), based on Ethan Gilmore’s original Zed theme port and customized with muted borders.

## Included themes

- **Kanagawa**
- **Kanagawa Wave**
- **Kanagawa Dragon**
- **Kanagawa Lotus**
- **Kanagawa - No Italics** variants
